# Akash Pawar — Portfolio

Static HTML/CSS/JS portfolio scaffold. Edit the markdown files in `content/` to update site content.

Local preview: open `index.html` in a browser.

Deploy to GitHub Pages (recommended):
1. Push repo to GitHub.
2. In repository settings > Pages, set source to `main` branch and `/ (root)` or use `gh-pages` branch.

Alternatively, use actions or deploy from `docs/` folder.
